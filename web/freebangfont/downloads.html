Free Bangla Fonts

Akaash (The Sky)
Akaash set of fonts developed by Sayamindu Dasgupta was the first font to be developed by our project using glyphs made by Prof. Palash Baran Pal. The current release (version 0.8.5)is still in a beta stage and has a number of known bugs and issues. If you download the font, please read the README & BUGS files and then please report any unknown bugs to our mailing lists.

MitraMono
MitraMono is an experimental monospace Bangla font developed by Dr Anirban Mitra for use in special applications like xterm and IE6. It also has additional ISCII encoding for Bengali.
The latest version 1.0.2 of the font can be downloaded from here |   Source Code
Mukti (Freedom)
One of the best fonts in our project is the Mukti set of fonts also developed by Dr. Anirban Mitra This font is developed using glyphs donated by Akruti Multimedia. Current version 3.0.2 is compliant with Unicode version 14.0 and updated OpenType specification version 2 for Bengali.

Installing the Fonts on Your System

To directly install this font in your Gnome desktop open the font using Gnome Font Viewer (Fonts) and press install.
If you use KDE Desktop, you an use Kfontview to view and install the fonts.
For Windows, right click on the .ttf file and select install

Alternately, if you are using a latest version of Debian or Ubuntu system, you can install our fonts easily by issuing this command from your console --
$ sudo apt-get install fonts-beng-extra
This will automatically install all our fonts in their latest version.

For Fedora linux you can install our fonts by issuing the followin commands
$ sudo dnf copr enable mitradranirban/fbf-fonts-all
$ sudo dnf install fbf-*-fonts
