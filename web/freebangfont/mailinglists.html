Our Mailing Lists

We have one mailing list for this project --

freebangfont-devel
This list is for discussions related to font development, administrative decisions, website management, Bug reports, etc. If you want to be actively involved in the development process, the consider joining this list. The posting address of the list is freebangfont-devel@nongnu.org.
The GNU Mailman listinfo page is at http://mail.nongnu.org/mailman/listinfo/freebangfont-devel. For subscribing to the list, please go to the listinfo page - there is a form for subscription/unsubscription at that page.
The archives for this list are at http://mail.nongnu.org/pipermail/freebangfont-devel/.
